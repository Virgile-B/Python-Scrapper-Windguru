API Project

The purpose of this API is to present information related to the conditions in a windsurfing spot called 'Le Porge'. Several tabs are available in this API in order to preview and manipulate the data. In the following sections, you will find a description of each tab.

Dashboard

The first tab retrieves the latest data that were web scrapped. Thus, it displays the quality of the windsurfding conditions of the latest days. A metric is associated with each period in a given day. This indicator is based on three factors: the temperature, the direction and the speed of the wind, which both affect the quality of the windsurfing conditions. Is it to be noted that the calculus of this metric i.e "Terrible" / "Bad" / ... is mapped in the config.py file and can be customized.

Data

This is the tab where you can look for data in a precise time window. You can also export a csv from the data that been web scrapped. It is to be noted that the filter also applies to data extracted through the csv. On top of that, an error would occur in case the format of the date is wrong. In the data section, you will find a table with the following parameters : [Date, Hour, Water Temperature, Wind Speed, Wind Direction]. Webscraping

This is where you can load and retrieve the latest data by executing the webscraping. This may take time to complete (~3 min).

Additional feature - SQL queries

As an additional feature, you are able to execute SQL queries in the test_sql.ipynb file. For instance, you are able to execute queries and filter the data based on some parameters such as the wind speed or the temperature.
